## 2025-05-15 - Empty State Call-to-Action
**Learning:** Users are often paralyzed by empty states ("No Items Found") if no immediate action is suggested. Providing a primary CTA button that matches the brand's main action (like the Floating Action Button) significantly reduces friction for first-time users or those with a cleared catalog.
**Action:** Always pair "Empty" or "Error" illustrations/messages with a primary action button if there is a logical next step the user should take.
